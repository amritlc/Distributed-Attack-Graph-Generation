# Distributed-Attack-Graph-Generation
The aim of this experiment is to reproduce the structure of Kaynar and Sivrikaya’s distributed attack graph generation experiment using a Python-based simulation on a 12 GB RAM laptop. The experiment will compare serial DFS with simulated distributed DFS using 2, 3, and 4 agents. 

Project:
Lightweight Simulation of Distributed Attack Graph Generation

Purpose:
A simplified Python simulation of the distributed attack graph generation approach proposed by Kaynar & Sivrikaya (2016). The implementation was developed to understand the algorithm, reproduce its core concepts on a resource-constrained laptop, and identify practical implementation challenges during the literature review.

Main Components:
- Synthetic network generation
- Reachability graph construction
- Network partitioning
- Serial DFS baseline
- Simulated distributed attack graph generation
- Performance evaluation
- Runtime visualisation

Environment:
Python 3.14
Windows 11
12 GB RAM
